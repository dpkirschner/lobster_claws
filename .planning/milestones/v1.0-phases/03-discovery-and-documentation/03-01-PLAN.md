       - name: "claws-cli"

       - import sys, importlib.metadata
       - `def discover_skills() -> dict[str, importlib.metadata.EntryPoint]`:
         Call `importlib.metadata.entry_points(group="claws.skills")`
         Return dict mapping skill name to entry point
       - `def main()`:
         If no args (len(sys.argv) < 2): list all discovered skills with name and description, exit 0
         If args[0] is a known skill: load the entry point and call it, passing remaining sys.argv
         If args[0] is unknown: print error to stderr with "Unknown skill: {name}. Available: {list}", exit 2
         IMPORTANT: Before calling the skill's main(), set sys.argv = [skill_name] + remaining_args so the skill's argparse sees the right argv
       - List format when no args: "Available skills:\n  transcribe  Audio transcription via whisper server\n" (get description from entry point metadata or hardcode pattern)
         Actually, use the entry point .value to show the module, or better: just list the names. Keep it simple: "Available claws:\n  transcribe\n\nUsage: claws <skill> [args...]"

    4. Create `cli/tests/__init__.py`: empty

       - Test main() with unknown skill prints error to stderr and exits 2
