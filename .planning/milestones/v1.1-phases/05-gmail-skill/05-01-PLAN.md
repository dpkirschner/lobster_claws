    **4. Create `gmail.py`** with these functions:

    Constants:
    ```python
    AUTH_PORT = 8301
    GMAIL_SCOPE = "https://www.googleapis.com/auth/gmail.modify"
    GMAIL_BASE = "https://gmail.googleapis.com/gmail/v1/users/me"
    ```

    Functions:
    - `get_access_token() -> str`: Create `ClawsClient("google-auth", 8301)`, call `post_json("/token", {"scopes": [GMAIL_SCOPE]})`, return `resp["access_token"]`.
    - `get_header(headers: list[dict], name: str) -> str`: Case-insensitive header lookup in Gmail's `[{"name": "...", "value": "..."}]` format. Return `""` if not found.
    - `extract_body(payload: dict) -> str`: Recursive MIME tree walk. Check `mimeType == "text/plain"` and `body.data` exists at current level. If multipart, recurse into `parts`. Return `""` if no text/plain found. Decode body data with `base64.urlsafe_b64decode(data).decode("utf-8")`.
    - `build_raw_message(to: str, subject: str, body: str, cc: str | None = None, bcc: str | None = None) -> str`: Use `email.mime.text.MIMEText(body)`, set To, Subject, optional Cc/Bcc headers. Return `base64.urlsafe_b64encode(msg.as_bytes()).decode("ascii").rstrip("=")`.
    - `_gmail_headers(token: str) -> dict`: Return `{"Authorization": f"Bearer {token}"}`.
    - `_gmail_get(path: str, token: str, params: dict | None = None) -> dict`: `httpx.get(f"{GMAIL_BASE}{path}", params=params, headers=_gmail_headers(token), timeout=30.0)`, raise_for_status, return json.
    - `_gmail_post(path: str, token: str, json_data: dict) -> dict`: Same pattern with httpx.post.
    - `_fetch_message_metadata(msg_id: str, token: str) -> dict`: Call `_gmail_get(f"/messages/{msg_id}", token, params={"format": "metadata", "metadataHeaders": ["From", "Subject", "Date"]})`. Extract headers, return `{"id": ..., "thread_id": ..., "from": ..., "subject": ..., "date": ..., "snippet": ...}`.
    - `list_inbox(max_results: int = 10) -> list[dict]`: Get token, call `_gmail_get("/messages", token, params={"q": "in:inbox", "maxResults": max_results})`, use `.get("messages", [])` for empty results safety, then `_fetch_message_metadata` for each ID. Return list of message dicts.
    - `read_message(msg_id: str) -> dict`: Get token, call `_gmail_get(f"/messages/{msg_id}", token, params={"format": "full"})`. Extract headers, call `extract_body(data["payload"])`. If body is empty, fall back to `data.get("snippet", "")`. Return `{"id": ..., "thread_id": ..., "from": ..., "subject": ..., "date": ..., "body": ..., "snippet": ...}`.
    - `send_message(to: str, subject: str, body: str, cc: str | None = None, bcc: str | None = None) -> dict`: Get token, call `build_raw_message(...)`, POST to `/messages/send` with `{"raw": encoded}`. Return `{"message_id": resp["id"], "thread_id": resp["threadId"]}`.
    - `search_messages(query: str, max_results: int = 10) -> list[dict]`: Get token, call `_gmail_get("/messages", token, params={"q": query, "maxResults": max_results})`, same metadata fetch pattern as list_inbox.
    - `handle_gmail_error(e: httpx.HTTPStatusError) -> None`: Parse error JSON, map 401->crash, 403->fail, 404->fail, 429->fail, else->crash. Import fail/crash from claws_common.output.

    **5. Run tests** and iterate until all pass.
