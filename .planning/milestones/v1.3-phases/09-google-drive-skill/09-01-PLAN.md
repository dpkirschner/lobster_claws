```python
AUTH_PORT = 8301
GMAIL_SCOPE = "https://www.googleapis.com/auth/gmail.modify"
GMAIL_BASE = "https://gmail.googleapis.com/gmail/v1/users/me"

def get_access_token(as_user: str | None = None) -> str:
    client = ClawsClient(service="google-auth", port=8301)
    body: dict = {"scopes": [GMAIL_SCOPE]}
    if as_user:
        body["subject"] = as_user
    resp = client.post_json("/token", body)
    return resp["access_token"]
```

Export MIME type mapping dict:

```python
EXPORT_MIME_TYPES = {
    "application/vnd.google-apps.document": "text/plain",
    "application/vnd.google-apps.spreadsheet": "text/csv",
    "application/vnd.google-apps.presentation": "application/pdf",
    "application/vnd.google-apps.drawing": "image/png",
}
```

Functions (all with `as_user: str | None = None`):

`get_access_token(as_user)` — Same pattern as gmail.py. POST to /token with DRIVE_SCOPE, optional subject.

`_drive_headers(token)` — Returns `{"Authorization": f"Bearer {token}"}`.

`_drive_get(path, token, params=None)` — GET to `DRIVE_BASE + path`, returns resp.json(). Timeout 30s.

`list_files(max_results=100, query=None, as_user=None) -> list[dict]`:
- Get token, GET /files with params:
  - `pageSize`: max_results
  - `fields`: `"files(id,name,mimeType,modifiedTime,size,parents)"`
  - `q`: query (only if provided)
  - `orderBy`: `"modifiedTime desc"`
- Return list of file dicts from response["files"]

`download_file(file_id, output_path, as_user=None) -> dict`:
- Get token
- Step 1: Fetch metadata via `_drive_get(f"/files/{file_id}", token, params={"fields": "id,name,mimeType,size"})`
- Step 2: Check if mimeType starts with "application/vnd.google-apps."
  - If Google Workspace doc: look up export MIME in EXPORT_MIME_TYPES. If not found, fail() with "Unsupported Google Workspace document type: {mimeType}". Use export URL: `GET {DRIVE_BASE}/files/{file_id}/export` with params `{"mimeType": export_mime}`. Response is binary.
  - If regular file: `GET {DRIVE_BASE}/files/{file_id}` with params `{"alt": "media"}`. Response is binary.
- Step 3: Write resp.content to output_path with open(output_path, "wb")
- Return `{"file_id": file_id, "path": output_path, "name": metadata["name"], "size": len(resp.content)}`
- Use timeout=120.0 for the download/export GET (larger files)

`upload_file(file_path, name, folder_id=None, as_user=None) -> dict`:
- Get token
- Read file bytes from file_path
- Detect MIME type: use `mimetypes.guess_type(file_path)[0] or "application/octet-stream"`
- Build metadata dict: `{"name": name}`. If folder_id: add `"parents": [folder_id]`
- Construct multipart/related body manually:
  ```python
  import json, uuid
  boundary = uuid.uuid4().hex
  body = (
      f"--{boundary}\r\n"
      f"Content-Type: application/json; charset=UTF-8\r\n\r\n"
      f"{json.dumps(metadata)}\r\n"
      f"--{boundary}\r\n"
      f"Content-Type: {mime_type}\r\n\r\n"
  ).encode("utf-8") + file_bytes + f"\r\n--{boundary}--".encode("utf-8")
  ```
- POST to `f"{DRIVE_UPLOAD_BASE}/files?uploadType=multipart"` with:
  - headers: `{"Authorization": f"Bearer {token}", "Content-Type": f"multipart/related; boundary={boundary}"}`
  - content=body (raw bytes, NOT json=)
  - timeout=120.0
- Return `{"id": resp_data["id"], "name": resp_data["name"], "mimeType": resp_data.get("mimeType", "")}`

`handle_drive_error(e: httpx.HTTPStatusError) -> None`:
- Same pattern as handle_gmail_error. Extract error message from response JSON.
- 401 -> crash("Drive authentication failed. Token may be expired or delegation misconfigured.")
- 403 -> fail(f"Drive access denied: {message}")
- 404 -> fail(f"File not found: {message}")
- 429 -> fail("Drive rate limit exceeded. Try again later.")
- else -> crash(f"Drive API error ({status}): {message}")
