         AUTH_PORT = 8301

       - `get_access_token(as_user=None)` -- requests BOTH scopes: [DOCS_SCOPE, DRIVE_READONLY_SCOPE]
       - `_docs_headers(token)` -- {"Authorization": f"Bearer {token}"}
       - `_docs_get(url, token, params=None)` -- GET returning JSON
       - `_docs_post(url, token, body)` -- POST returning JSON
       - `extract_text(document: dict) -> str`:
         ```python
         content = document.get("body", {}).get("content", [])
         parts = []
         for element in content:
             paragraph = element.get("paragraph")
             if paragraph:
                 for elem in paragraph.get("elements", []):
                     text_run = elem.get("textRun")
                     if text_run:
                         parts.append(text_run.get("content", ""))
         return "".join(parts)
         ```
       - `list_documents(max_results=100, as_user=None) -> list[dict]`:
         GET {DRIVE_BASE}/files with q="mimeType='application/vnd.google-apps.document'", pageSize, fields="files(id,name,modifiedTime)"
       - `read_document(doc_id: str, as_user=None) -> dict`:
         GET {DOCS_BASE}/{doc_id}, extract text, return {"documentId": doc["documentId"], "title": doc["title"], "text": extract_text(doc)}
       - `create_document(title: str, body: str | None = None, as_user=None) -> dict`:
         POST {DOCS_BASE} with {"title": title} to create blank doc.
         If body is provided: POST {DOCS_BASE}/{doc_id}:batchUpdate with:
         {"requests": [{"insertText": {"text": body, "endOfSegmentLocation": {}}}]}
         Return {"documentId": ..., "title": ...}
       - `append_text(doc_id: str, text: str, as_user=None) -> dict`:
         POST {DOCS_BASE}/{doc_id}:batchUpdate with:
         {"requests": [{"insertText": {"text": text, "endOfSegmentLocation": {}}}]}
       - `handle_docs_error(e)` -- same pattern as other skills
